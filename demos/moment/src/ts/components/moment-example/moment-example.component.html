Formatters

toMoment

For this demo we assign two new variables to the model called pastDate and futureDate:
rv-assign-past-date="'2019-09-15T15:00:00.000Z' | toMoment"
rv-assign-future-date="'2022-09-10T16:00:00.000Z' | toMoment"

Now we can work with these variables, e.g. to convert pastDate to a vanilla date:

{ pastDate | toDate }
{ pastDate | toDate }
